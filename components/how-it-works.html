How DiSCon-X Works

Our comprehensive security solution integrates mobile protection with DICT's whitelist verification system

Click on the diagram to zoom in, drag to pan, and use the controls to adjust the view

1

Network Scanning

DiSCon-X constantly monitors surrounding Wi-Fi networks, collecting data about signal strength, security protocols, and network identifiers.

2

Whitelist Verification

Each detected network is checked against DICT's regularly updated whitelist of verified public Wi-Fi access points.

3

Threat Analysis

Advanced algorithms detect inconsistencies and suspicious behaviors that might indicate an evil twin attack or other network security threats.

4

User Alerts & Protection

Users receive instant notifications about potential threats with recommendations on secure networks to use instead.
